# Crazyflie-Christmas-Tree
A repository for creating a Christmas tree choreography with a swarm of Crazyflies.

## Ingredients
 - 8 Crazyflie drones
 - 8 Color LED decks (bottom-mounted)
 - 8 Lighthouse positioning decks
 - 1 Lighthouse positioning system
 - 3 Crazyradios 2.0

 ## Process
 - Modify the [firmware](/firmware/) and flash it to all Crazyflies
 - Run the [python script](/python_script/)
